🌍 AI Trip Planner
A smart travel planning web app built with React and AI that creates personalized trip itineraries based on user preferences like budget, duration, and interests.

🚀 Features
✈️ AI-generated travel plans
🗓️ Day-wise itinerary
💰 Budget-based suggestions
📍 Destination recommendations
📱 Responsive UI

🧠 How It Works
Enter trip details (destination, days, budget, interests)
AI generates a customized itinerary
View structured day-by-day travel plan

🛠️ Tech Stack
Frontend: React.js
AI: OpenAI / Gemini API
Routing: React Router
State: useState / Context API

⚙️ Setup
git clone https://github.com/your-username/ai-trip-planner.git
cd ai-trip-planner
npm install
npm start

Create a .env file:

REACT_APP_AI_API_KEY=your_api_key
